# Automação E2E com Cypress - E-commerce

## 📌 Sobre
Este projeto demonstra uma estratégia de teste ponta a ponta para um aplicativo de e-commerce.

## 🧪 Estratégia de Teste
- Foco em jornadas críticas do usuário
- Cenários positivos e negativos
- Simulação de falhas por meio de mocks de API

## 🚀 Tecnologias
- Cypress
- JavaScript
- Mochawesome

## 🎯 Fluxos Cobertos
- Login
- Adicionar ao carrinho
- Finalizar compra
- Compra

## ⚠️ Não Cobertos
- Testes de desempenho
- Testes de segurança

## 🔁 CI/CD
Os testes são executados automaticamente usando o GitHub Actions.
